# UIF Quasar Cosmology Experiment — Output Directory

This folder contains all processed outputs from the UIF cosmology-lite emulator
used in the quasar cosmology experiments (Experiments I–V). These outputs were
generated by running the scripts in `/code/quasar_cosmology_experiment/` using
the merged SDSS Stripe 82 quasar dataset stored in
`/data/quasar_cosmology_experiment/`.

All files in this directory are *derived* products: figures, calibration tables,
diagnostics, κ-maps, hysteresis traces, threshold maps, and operator summaries.

No raw SDSS data are redistributed here. See the data folder README for details.

The structure mirrors the top-level folder:
- gamma_sweep.csv  
- threshold_map.csv

---

## 3. How These Outputs Were Generated

All files were produced by running:
- python ut26_cosmo3d.py
- python plot_gamma_sweep.py
- python plot_threshold_heatmap.py
- python plot_hmf.py
- python plot_kappa_map.py

with the following configuration:

- **Dataset:** SDSS Stripe 82 (merged, cleaned)  
- **Emulator:** UIF cosmology-lite (96³–128³ lattice)  
- **Timesteps:** T = 300–700  
- **Default parameters:**  
  - β = 3.0  
  - λᴿ ≈ 0.20  
  - η* ≈ 0.55  
  - Γ₀ ≈ 0.9  

The outputs include:
- Coherence fields  
- κ-maps  
- Power spectra  
- Γ-sweep diagnostics  
- Threshold-region maps  
- Logistic fits  
- Operator summaries (ΔI, Γ, λᴿ, R∞, k)
