## Reproducibility

All results here are generated from scripts in:

UIF/code/cosmology_lite_experiment/

Inputs come exclusively from the emulator (no external datasets).  
To regenerate the outputs:

1. Run baseline:
   python ut26_cosmo3d.py

2. Run operator sweeps:
   python run_gamma_sweep.py
   python run_threshold_map.py

3. Generate figures:
   python plot_gamma_2x2.py
   python plot_gamma_sweep_heatmap.py
   python plot_threshold_heatmap.py
   python ut26_cosmo3d_hysteresis.py

All scripts are deterministic given the random seed set in each file.
