# EEG Coherence Experiment Data

This folder contains raw and processed datasets supporting the *UIF Companion Experiments (2025)* and *UIF VII – Predictions and Experiments*.

## Description
EEG datasets measuring cross-domain coherence under the CHREM/RSIPP protocol.
- Participants: N = [number]
- Channels: [EEG channels used]
- Sampling rate: [Hz]
- Conditions: Eyes-open / eyes-closed, left/right hemisphere integration, meditative state protocols.
- Recorded variables: voltage traces, power spectral densities, coherence indices.

## Files
- `raw/` — Original EEG export files (e.g., .edf, .csv)
- `processed/` — Preprocessed and band-filtered data
- `analysis/` — Derived coherence indices, ΔI, λᴿ, Γ time-series
- `figures/` — Visualisations for UIF VII
- `metadata.json` — Run-tagged info, software version, preprocessing notes
