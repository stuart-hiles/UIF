# Quasar Cosmology Experiment Data

This folder contains synthetic and observational datasets used in the cosmology-lite emulator validation (Experiments I–V).

## Description
Operator calibration using quasar light-curve and cosmological coherence data.
- Emulator outputs: `ut26_cosmo3d_outputs/`
- Observational cross-checks: JWST / EHT / DESI / Planck residuals
- Variables: mean coherence ⟨s⟩, cumulative pruning, P(k), κ-power, R∞, k, λᴿ, η*.

## Files
- `ut26_cosmo3d_outputs/` — Raw emulator outputs (HDF5, CSV)
- `derived/` — Processed operator fits and calibration tables
- `figures/` — All figure PNGs referenced in the UIF Companion paper
